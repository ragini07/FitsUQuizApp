
# KidzWorld


## Built using

- ReactJS
- Context API and Reducer
- React Router
- CSS from my Component Library [FitsU-UI](https://fitsu-ui-design.netlify.app/)


## Features

* Home Page with categories
* Quizzes under category
* Rules Page
* MCQ based quiz questions
* Score and result page
* Quit quiz
* Timer per questions
* Responsive
* Dark Mode
